# تحديثات منصة كرم - ديسمبر 2025

## الملفات المُضافة

### صفحات HTML الجديدة
1. **family-details.html** - صفحة تفاصيل الأسرة المستضيفة
   - معرض صور تفاعلي
   - عرض الباقات المتاحة
   - نظام الحجز المباشر
   - عرض التقييمات والمراجعات

2. **cart.html** - صفحة سلة الحجوزات
   - عرض الحجوزات المؤقتة
   - ملخص الأسعار
   - نظام كوبونات الخصم
   - الانتقال المباشر للدفع

3. **visitor-dashboard.html** - لوحة تحكم المعتمر
   - إدارة الحجوزات (القادمة والسابقة)
   - المفضلة
   - تحديث الملف الشخصي
   - الإشعارات

4. **company-dashboard.html** - لوحة تحكم الشركات (B2B)
   - إدارة الحجوزات الجماعية
   - عرض الفواتير
   - التقارير والإحصائيات
   - إعدادات الشركة

### ملفات JavaScript
1. **js/family-details.js** - إدارة صفحة تفاصيل الأسرة
2. **js/visitor-dashboard.js** - إدارة لوحة تحكم المعتمر
3. **js/company-dashboard.js** - إدارة لوحة تحكم الشركات

### ملفات CSS
1. **styles/family-details.css** - تنسيقات صفحة التفاصيل
2. **styles/cart.css** - تنسيقات صفحة السلة

### ملفات التهيئة
1. **.env.example** - قالب متغيرات البيئة
2. **.gitignore** - تحديث لاستبعاد ملفات النسخ الاحتياطي

## الميزات الجديدة

### 1. نظام التفاصيل الكامل للأسرة
- معرض صور احترافي مع modal
- عرض المميزات والخدمات
- نظام حجز مباشر
- عرض التقييمات مع ردود الأسر

### 2. سلة تسوق متقدمة
- إدارة الحجوزات المتعددة
- حساب تلقائي للأسعار
- نظام كوبونات الخصم
- واجهة بسيطة وسهلة الاستخدام

### 3. لوحات تحكم شاملة
- لوحة تحكم المعتمر مع جميع الحجوزات
- لوحة تحكم الشركات للحجوزات الجماعية
- إحصائيات وتقارير تفصيلية
- إدارة كاملة للملف الشخصي

## التحسينات

### الأداء
- تحميل ديناميكي للبيانات
- تحسين استعلامات Supabase
- Lazy loading للصور

### تجربة المستخدم
- واجهات سلسة ومتجاوبة
- رسائل توضيحية واضحة
- تنقل سهل بين الصفحات

### الأمان
- فصل ملفات التهيئة
- حماية API Keys
- Row Level Security في Supabase

## التكامل مع الملفات الموجودة

جميع الملفات الجديدة تتكامل بسلاسة مع:
- نظام التصميم الموجود (`design-system.css`)
- ملفات JavaScript الأساسية (`main.js`, `auth.js`, `cart.js`)
- قاعدة البيانات الكاملة (`complete_schema.sql`)
- نظام Moyasar للمدفوعات

## الخطوات التالية

### للتطوير المحلي
1. نسخ `.env.example` إلى `.env`
2. تحديث مفاتيح Supabase و Moyasar
3. تشغيل خادم محلي (Live Server)

See `DEPLOYMENT.md` for deployment instructions
